Handwritten Character Recognition

Project Overview

This project recognizes handwritten digits using Machine Learning techniques. The model is trained using the Digits dataset from Scikit-Learn and implemented using the Random Forest algorithm.

Technologies Used

- Python
- Scikit-Learn
- Machine Learning
- Random Forest

Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

Results

- Accuracy: 97.22%
- Predicted Digit: 6
- Actual Digit: 6

Conclusion

The model successfully recognizes handwritten digits with high accuracy and demonstrates the practical application of Machine Learning in character recognition.
